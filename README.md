# US-Air-Pollution-Analytics-Dashboard-2006-2010-
A multi-page Power BI dashboard analyzing four major air pollutants — Carbon Monoxide (CO), Ozone (O3), Sulphur Dioxide (SO2), and Nitrogen Dioxide (NO2) — across US states and counties from 2006 to 2010. The dashboard uses AQI measurements to identify the most polluted states, track yearly trends, and support environmental data exploration.
